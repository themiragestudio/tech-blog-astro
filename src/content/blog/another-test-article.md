# 第二篇测试文章

这是第二篇测试文章，用于验证博客的导航功能。

## 导航功能测试

博客应该支持以下导航功能：

### 1. 上一篇/下一篇导航
在文章详情页底部，应该显示上一篇和下一篇文章的链接。

### 2. 面包屑导航
显示当前文章的路径，方便用户返回。

### 3. 相关文章
基于标签或分类推荐相关文章。

## 功能验证清单

- [x] 文章列表页显示所有文章
- [x] 文章详情页正确渲染内容

## 性能考虑

这个博客项目注重性能：

1. **静态生成** - 无运行时开销
2. **代码分割** - 按需加载 JavaScript
3. **图片优化** - 自动生成响应式图片
4. **CSS 优化** - 使用 Tailwind CSS 进行 tree-shaking
